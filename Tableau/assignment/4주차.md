## 33. 계산된 필드

1. 데이터 패널을 통해 생성
2. 분석 탭을 통해 생성
3. 사용하고자 하는 필드 위에 마우스 우클릭 하여 생성 

**작성법**   
필드이름: [필드이름]    
문자값: '문자값'

**계산된 필드 만들기**

2. 이름 지정(ex. 수익률)
3. 함수식 작성 (ex. SUM([수익])/SUM([매출]))  
4. 계산된 마우스 필드 우클릭 -> 기본 속성 변경

## 34. 행수준계산

<!-- 행수준 계산의 의미와 적용 방법을 적어주세요. -->
계산된 필드(기본 계산/테이블 계산/LOD 표현식)   
 **기본 계산**
 : 데이터 원본에 대한 행 수준 계산 또는 집계 계산   
 행 수준 계산: 데이터의 각 레코드를 통해 계산   
 집계 계산: 현재 뷰에서 보이는 기준으로 계산   

- 고객 이름을 성/이름으로 분리하기
```sql
SPLIT([고객이름], ' ', 1) 
--SPLIT 함수: SPLIT(문자열, 구분 기호, 토큰 번호)   
--고객 이름 필드와 계산된 필드를 행에 Drag and Drop 
--고객의 성이 필요한 경우 -> 
SPLIT([고객이름], ' ', 2) 
 ```

- 수익성 필드 만들기   
1. 계산된 필드 만들기 선택
2. 계산식 작성 
```sql
IF[수익]>0 THEN '수익성 있음' 
ELSEIF[수익] <0 THEN '수익성 없음'
ELSE '손익 평형'
END 
```
- 두 날짜 사이 일 수 구하기   
```sql
DATEDIFF('day', [주문 날짜], [배송날짜])
```
측정값 -> 차원

## 36. 테이블계산

 


<!-- 테이블 계산의 개념 및 사용 방법을 적어주세요. -->
테이블 계산: 뷰에 보이는 내용을 바탕으로 데이터 계산   
- 월마다 연도 따라 누적 매출 찾기   
-> RUUNING_SUM(SUM([매출]))   

- 계산 방향 변경하기  
1. 필드 마우스 우클릭 -> 다음을 사용하여 계산   
2. 필드 마우스 우클릭 -> 테이블 계산 편집

- 현재 달과 전 월의 차이 알아보기   
-> SUM([매출])-LOOKUP(SUM([매출]),-1)
## 37. 퀵테이블계산(1)

<!-- 퀵테이블 계산의 원리 및 예제에 대해 알게 된 점을 적어주세요. -->
퀵테이블계산: 테이블 계산에서 가장 자주 쓰이는 계산 유형들을 클릭만으로 가능하도록 만들어놓은 기능   

사용하고자하는 필드 위에 마우스 우 클릭 -> "퀵 테이블 계산" 선택   

- 누계: 집계한 값을 누적한 값으로 한 번 더 집계   

**이중축 이용하여 누계 나타내기**
1. 복제하여 만든 필드 마우스 우클릭->누계 선택
2. 이중축 적용

**퀵 테이블 계산의 차이를 통해 각 제품에 대한 연도별 매출 차이 알아보기**
1. 매출 필드 마크-텍스트 
2. 매출 필드 마우스 우클릭 -> "퀵 테이블 계산" -> "차이"
3. 기준값 옵션 지정

**비율 차이**
1. 수익 필드 기호 클릭 -> "퀵 테이블 계산" -> "비율 차이"

**지역별 제품 매출 순위와 구성비율**
1. 마크 선반의 "매출" 필드 기호 클릭 -> "퀵 테이블 계산" -> "순위"
2. 매출 필드 세모 클릭 -> "다음을 사용하여 계산" -> "패널(옆에서 아래로)
3. 측정값 선반 필드 기호 클릭 -> "퀵 테이블 계산" -> "구성비율" 
4. 다음을 사용하여 계산 -> "패널(아래로)"

**백분위수**
1. 마크 선반 매출 필드 마우스 우클릭 
2. "퀵 테이블 계산" -> "백분위수"

## 38. 퀵테이블계산(2)

<!-- 이동평균, YTD 총계, 전년 대비 성장률, YTD 성장률 등 본 강의에서 알게 된 점을 적어주세요. -->
**이동평균**   
: 이전의 값부터 현재의 값에 대한 평균을 낼 때 사용   

1. 매출 필드 Control 복사
2. 마우스 우클릭 -> '퀵테이블 계산' -> '이동평균'
3. 매출 필드 마우스 우클릭 -> 테이블 계산 편집
4. 이중축 클릭

**YTD(Year to Date) 총계**    
: 특정 시점을 기준으로 해당 연도부터 그 시점까지 총계   
'분기' 또는 '월'이 있어야 사용 가능   

 1. 마크카드 '매출' 필드 마우스 우클릭 
 2. '퀵 테이블 계산' -> 'YTD 총계'
 
**통합 성장률(Compound Growth Rate)**   

1. 마크카드 '매출' 필드 마우스 우클릭  
2. '퀵 테이블 계산' -> '통합 성장률'

**전년 대비 성장률(Year Over Year Growth)**   
: 같은 월을 기준으로 이전 연도 대비 얼마정도 성장했는지 나타냄   

1. 마크카드 '매출' 필드 마우스 우클릭
2. '퀵 테이블 계산' -> '전년 대비 성장률'

**YTD 성장률(Year to Date Growth)**  

1. 마크카드 '매출' 필드 마우스 우클릭
2. '퀵 테이블 계산' -> 'YTD성장률'
3. 측정값 카드 합계(매출) 데이터 패널로 이동
4. '계산 1' -> 'YTD 총계'로 변경
5. '계산 1 1' -> 'YTD 성장률'로 변경
6. 'YTD 총계' 필드 측정값 가운데 넣기

## 문제 2.

이때, 계산된 필드의 식은 'DATEDIFF' 함수를 이용하였습니다.

배송까지 걸린 일수 계산을 위한 DATEDIFF 함수 수식을 적어주세요.

```
DATEDIFF('day', [주문 날짜], [배송날짜])
```

## 문제 3.

다음은 Tableau의 다양한 계산을 사용할 수 있는 경우를 빈칸으로 두고 문제를 작성한 것입니다. 각 빈칸에 적합한 계산 유형을 채워보세요.

보기
> **누계, 차이, 비율 차이, 구성 비율, 순위, 백분위수, 이동 평균, YTD 총계, 통합 성장률, 전년 대비 성장률, YTD 성장률**

| 계산 유형               | 설명                                                                 | 사용 예시                                                                                          |
|-------------------------|----------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| 누계           | 데이터의 누적 합계를 계산                                             | 한 기업이 월별 매출 데이터를 누적하여 연간 매출 추이를 보고 싶을 때 사용                                      |
| 차이            | 연속 데이터 포인트 간의 차이를 계산                                    | 한 기업이 월별 매출 데이터에서 전월 대비 매출 증감량을 분석하고 싶은 경우                                        |
| 비율 차이           | 연속 데이터 포인트 간의 비율 변화를 계산                               | 한 기업이 월별 매출 데이터에서 전월 대비 매출 증감률(%)을 분석하고 싶은 경우                                      |
| 구성 비율            | 전체에서 각 데이터 포인트의 비율을 계산                                | 한 기업이 전체 매출에서 각 제품군이 차지하는 비율을 보고 싶을 때 사용                                           |
| 순위            | 데이터의 순위를 매깁니다                                              | 한 기업이 제품별 매출 데이터를 순위별로 정렬하여 상위 10개 제품을 분석하고 싶은 경우                              |
